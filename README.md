## Installation
To install and run the QR-CODE-Generator project, follow these steps:
```bash
# Clone the repository
git clone https://github.com/Tusharkamthe23/QR-CODE-Generator.git

# Navigate to the project directory
cd QR-CODE-Generator

# Install dependencies
pip install -r requirements.txt
```

## Usage
The `QR_code_genrator.py` script can be executed from the command line, accepting parameters such as the QR code's contents, size, and error correction level.
```python
# Example usage
python QR_code_genrator.py -c "https://www.example.com" -s 200 -e "H"
```
